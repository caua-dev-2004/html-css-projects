# Projeto Final - Principais Linguagens de Programação

Este é o projeto sobre as principais linguagens de programação. O projeto tem como objetivo apresentar informações sobre as linguagens de programação mais populares no mercado, com conteúdo detalhado sobre cada uma delas, incluindo HTML5, PHP, Java, C#, Python, C++, SQL e JavaScript.

## Descrição

O projeto é um site que contém informações sobre linguagens de programação populares e oferece um formulário de cadastro. O site é estruturado em HTML5 e CSS3, com links para cada seção das linguagens de programação, além de um formulário para coletar informações dos usuários.

### Funcionalidades:
- **Navegação**: Links para cada linguagem de programação, permitindo que o usuário acesse diretamente a descrição de cada uma.
- **Formulário de Cadastro**: Permite que os usuários preencham informações pessoais, como nome, data de nascimento, e-mail, telefone e preferências de cursos.
- **Design Responsivo**: A interface foi projetada com foco na usabilidade e acessibilidade.

## Tecnologias Utilizadas

- **HTML5**: Estrutura principal do site.
- **CSS3**: Estilos de layout e design, criando uma interface visualmente atraente.
